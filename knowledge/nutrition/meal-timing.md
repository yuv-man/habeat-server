# MEAL TIMING × CIRCADIAN BIOLOGY

## circadian_rhythm_basics
cortisol_peak: `7–9am` → natural appetite + insulin_sensitivity peak → best window for largest meal
melatonin_onset: `~9pm` → ↓ insulin_secretion → ↓ glucose_tolerance → fat_storage preference ↑
metabolic_window: calories_consumed `before 3pm` → better weight outcomes than same calories after
thermic_effect: same meal eaten at breakfast → 2.5× higher thermic_effect than at dinner

## breakfast
skip_breakfast_effects:
  → compensatory_overeating later in day (+20% total_daily_intake on avg)
  → ↑ ghrelin (hunger hormone) spike mid-morning → cravings, irritability
  → ↓ cognitive performance, ↓ impulse_control by 11am
  → ↑ emotional_eating_risk in evening (OR 1.6)
optimal_breakfast: high_protein (`25–35g`) + complex_carbs + healthy_fat
  → stable_glucose 4–5h | ↓ afternoon_snacking by 31%
  → tryptophan + carb_co-ingestion → serotonin primed for day

## meal_frequency
3–4_meals_per_day: optimal glucose_stability for most non-athletes
  → `3.5–4h` gaps → allows insulin to return to baseline
  → prevents reactive_hypoglycemia (glucose dip → cortisol spike → emotional_eating trigger)
grazing_risk: continuous eating → insulin_chronically_elevated → leptin_resistance → satiety signal blunted
long_gaps_risk: `>5h` without food → glucose depletion → amygdala_reactivity ↑ → impulse eating ↑

## late_night_eating
risk_window: `after 9pm` (or within 2h of sleep)
mechanisms:
  melatonin ↑ → insulin_secretion ↓ → glucose_clearance slowed → fat_storage ↑
  core_body_temp ↓ → digestive_motility ↓ → discomfort, poor_sleep quality
  sleep_disruption → ghrelin ↑ + leptin ↓ → hunger↑ next day → ee_cycle perpetuated
ee_correlation: late_night_eating_frequency correlates r=0.61 with stress_eating_score
intervention: kitchen_closed_time ritual → removes_decision → reduces_ee by ~40% at that window
exception: small protein_snack (`15–20g`) for hypoglycemia-prone individuals or high exercise load

## intermittent_fasting
16:8 protocol: eating_window `8am–4pm` or `10am–6pm` → aligns with circadian peak
benefits: insulin_sensitivity ↑, autophagy, ↓ inflammatory_markers
ee_risk: prolonged_fasting → hunger_amplifies_emotional_triggers → ↑ binge_risk
  recommendation: not_suitable for active emotional_eaters or restriction-binge_cycle users
  safer_alternative: time_structured_meals (3 set times) without strict fasting window

## meal_size_distribution
optimal_distribution:
  breakfast: 25–30% daily_calories
  lunch: 35–40% daily_calories
  dinner: 25–30% daily_calories
  snacks: 5–10% daily_calories
evening_heavy_pattern: >40% calories after 6pm → ↑ ee_risk, ↑ weight_gain_risk, ↓ sleep_quality
